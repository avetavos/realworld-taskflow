ยินดีต้อนรับ ในคอร์สนี้คุณจะได้สร้าง **TaskFlow** บอร์ด Kanban แบบเรียลไทม์ที่รองรับผู้ใช้หลายคน ตั้งแต่ repo เปล่า ๆ ไปจนถึงสแตกที่รันได้ด้วย `docker compose up` — ฝั่ง back end ใช้ Rust/Axum + PostgreSQL + Redis, ฝั่ง front end ใช้ Astro + Preact island และเชื่อมทุกอย่างเข้าด้วยกันด้วยการซิงก์แบบเรียลไทม์ผ่าน WebSocket
